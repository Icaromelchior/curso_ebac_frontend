Tag (nav): É comum incluir menus de navegação no header
para permitir que os usuários acessem facilmente outras
partes do site.

A tag (div) é usada para criar uma seção genérica em uma página da web.
Ela é frequentemente usada para agrupar elementos HTML e aplicar estilos
CSS a eles. Você pode adicionar qualquer conteúdo desejado entre as tags de
abertura (div) e de fechamento (/div). Por exemplo, você pode adicionar texto,
imagens, links, formulários ou outros elementos HTML dentro de uma div.

Tag (main) Conteúdo principal da página

Section é usado para representar uma seção ou um agrupamento
temático de conteúdo em uma página da web. Ele ajuda a estruturar
o conteúdo da página e a criar uma hierarquia semântica. O elemento (section)
é geralmente usado quando você deseja dividir o conteúdo da página em seções
distintas e significativas.

(article) envolve o conteúdo principal do artigo.
