Formulários em HTML são elementos que permitem aos usuários inserir
informações e enviar dados para um servidor web ou manipulá-los localmente
no navegador. Eles são criados usando uma combinação de elementos HTML,
como (form /form), (input), (textarea), (select), entre outros. Aqui está uma
explicação básica de como criar formulários em HTML:

1.(form): O elemento (form) é usado para criar um formulário HTML. Ele contém
todos os elementos de entrada e define para onde os dados do formulário serão
enviados quando o usuário o enviar.

(form) action="processar_dados.php" method="post"

(/form)

action: O atributo "action" define a URL ou o script para onde os dados do formulário
serão enviados para processamento. method: O atributo "method" especifica o método HTTP a ser
usado para enviar os dados, geralmente "post" ou "get".

2.(input): O elemento (input) é usado para criar campos de entrada, como campos de texto, caixas de
seleção, botões de rádio e caixas de seleção.

type: O atributo "type" especifica o tipo de campo de entrada, como "text", "password",
"submit", "radio", "checkbox" e outros.
name: O atributo "name" é usado para identificar o campo de entrada quando o formulário é
enviado para o servidor.

2.(textarea): O elemento (textarea) é usado para criar campos de texto multilinha.



(select) e (option): O elemento (select) cria uma lista suspensa (menu de seleção), e (option) define as opções dentro dessa lista.


3.(button): O elemento (button) cria um botão que pode ser usado para enviar o formulário
ou executar alguma ação JavaScript.


Estes são os elementos básicos para criar um formulário em HTML. Você pode
adicionar estilos CSS para personalizar a aparência do formulário e usar JavaScript
para adicionar funcionalidades interativas, como validação de dados do formulário.
Quando o usuário preencher o formulário e clicar no botão "Enviar," os dados serão
enviados para o servidor especificado no atributo "action" do elemento (form).
