Tabelas são elementos importantes em HTML que permitem
organizar e exibir dados de forma tabular. Elas são criadas
usando uma combinação de elementos HTML, como (table), (tr), (th),
e (td). Aqui está uma explicação básica de como criar tabelas em HTML:

1. table: O elemento table é usado para criar uma tabela. Ele é o elemento
pai de toda a estrutura da tabela.

Conteúdo da tabela aqui

2.(tr): O elemento (tr) é usado para criar uma linha na tabela. Cada linha deve ser inserida dentro de uma tabela (table).

Conteúdo da primeira linha

Conteúdo da segunda linha

3.(td): O elemento (td) é usado para criar células de dados na tabela.
As células de dados contêm o conteúdo real da tabela.

Nome Idade
João 30
Maria 25

Além desses elementos básicos, você também pode usar atributos para
personalizar a aparência e o comportamento da tabela, como border,
cellspacing, cellpadding, e muitos outros.
