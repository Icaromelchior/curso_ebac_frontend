Listas ordenadas

Em HTML, você pode criar listas de diferentes tipos: listas ordenadas,
listas não ordenadas e listas de definição. Abaixo estão exemplos de como
criar cada tipo de lista:
Lista Ordenada (Ordered List):Uma lista ordenada é uma lista em que os itens
são numerados em ordem. Use a tag (ol) para criar uma lista ordenada e a tag (li)
para cada item da lista.

  1. Item 1
  2. Item 2
  3. Item 3

Lista Não Ordenada (Unordered List):

Uma lista não ordenada é uma lista em que os itens são
marcados com símbolos, como pontos ou traços. Use a tag (ul) para criar uma lista não ordenada
e a tag (li) para cada item da lista.

  • Item A
  • Item B
  • Item C

Lista de Definição (Definition List):

Uma lista de definição é uma lista em que cada item é composto
por um termo (uma palavra ou frase) seguido de sua definição.
Use as tags (dl) para criar uma lista de definição, (dt) para
os termos (definições de título) e (dd) para as definições
propriamente ditas.
